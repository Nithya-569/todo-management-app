Create a full-stack Todo Management Application where users can sign up, log in, and manage multiple Todo Lists.

Glossary:
Todo Item: A task to be done

Todo List: A collection of Todo Items

These are the capabilities the app must have:

Users can sign up and log in

Create and Manage Todo Lists
- Users can create multiple Todo Lists
- Users can rename and delete Todo Lists
- Users should be able to access their Todo Lists from any device

Manage Todo Items within Todo Lists
- Create Todo Items
- Mark Todo Items as completed / not completed
- Rename / delete Todo Items
- Assign tags to Todo Items eg: urgent, not important, etc
- Filter Todo Items by tags

Users can view these stats per Todo List:
- Completed / Pending / Total Todo Items
- Number of Todo Items in each tag eg: urgent (2), not important (1), low priority (5), no tag (5)

Users can generate a unique link for their Todo List, which allows anyone with the link to view the Todo List
- Publicly shared Todo Lists should be clearly marked

Tech Stack Used: 
frontend : React and TailwindCSS Configuration
Backend: Node.js and Express.js
